# bpg-ap-pega-ui-framework-poc


### Installation requirements
for Python 2.7:
```
pip install --upgrade selenium
pip install --upgrade robotframework
pip install --upgrade robotframework-seleniumlibrary
pip install --upgrade robotframework-pageobjectlibrary
pip install exchangelib
```

For Python 3:
```
pip3 install --upgrade selenium
pip3 install --upgrade robotframework
pip3 install --upgrade robotframework-seleniumlibrary
pip3 install --upgrade robotframework-pageobjectlibrary
pip3 install exchangelib
```

Chromedriver
```
Download chromedriver from http://chromedriver.chromium.org/downloads
Add location of downloaded chromedriver to environment path
```

for Eclipse user:
```
1. Download Eclipse IDE from https://www.eclipse.org/downloads/ then instal
2. Open Eclipse, then go to HELP -> Eclipse Marketplace
3. Search for RED-Robot Editor. Once found install the plugin. 
4. Open Eclipse, then go to HELP -> Eclipse Marketplace
5. Search for PyDev. Once found install the plugin. 
6. Restart Eclipse
7. Import project either via file or git
```
